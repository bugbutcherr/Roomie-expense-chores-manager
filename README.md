# Roomie

Something i built to get a hang of HTML and js
## Features

* **Expense Splitting:** Easily log common expenses like rent, groceries, and utilities, and split them accurately among roommates.
* **Debt Tracking:** Keep a clear, ongoing record of who owes whom, avoiding confusion and awkward conversations.
* **Chore Management:** Assign and track household chores to ensure everyone contributes equally to maintaining the living space.
* **Monthly Settlement Summaries:** Generate clear, concise summaries at the end of the month to settle all outstanding balances effortlessly.
* **UPI Integration:** Seamlessly integrate UPI-based reminders for quick and easy settlements.
* **Single-File Architecture:** The entire application is contained within `roomie.html`, making it incredibly easy to deploy, share, and use without any complex server setups or build steps.

## Getting Started

1. Clone this repository or simply download the `roomie.html` file.
2. Open `roomie.html` in any modern web browser.
3. Start managing your shared living expenses and chores!
